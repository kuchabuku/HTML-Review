# 基础

## 标题

```html
<h1>这是标题 1</h1>
<h2>这是标题 2</h2>
<h3>这是标题 3</h3>
<h4>这是标题 4</h4>
<h5>这是标题 5</h5>
<h6>这是标题 6</h6>
```

## 段落

```html
<p>这是一个段落。</p>
<p>这是一个段落。</p>
<p>这是一个段落。</p>
```

## 折行

```html
<p>这个<br>段落<br>演示了分行的效果</p>
```

## 链接

```html
<a href="https://www.baidu.com">这是一个链接使用了 href 属性，链接到百度</a>
```

## 图像

```html
<img src="/images/logo.png" width="100" height="100" />
```

## 水平线

```html
	<p>这是段落。</p>
	<hr />
	<p>这是段落。</p>
```

## 注释

```html
<!--这是一个注释，注释在浏览器中不会显示-->
<p>这是一个段落</p>
```
